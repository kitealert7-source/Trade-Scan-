ENTRY LOGIC (UNION)

A LONG entry is triggered when position is FLAT and ANY ONE of the following conditions is true on the current bar:

1. Three Consecutive Lower Closes

Close < Close(T-1)

Close(T-1) < Close(T-2)

Close(T-2) < Close(T-3)

2. Stochastic Oversold Cross

Stochastic %K (14, 3)

%K < 20

%K(T-1) ≥ 20

3. ROC Oversold Cross

ROC(5) < -1.0%

ROC(5)(T-1) ≥ -1.0%

4. RSI Deep Oversold

RSI(2) calculated

Average of RSI(T-1) and RSI(T-2) ≤ 25

EXIT LOGIC (STRICT PRECEDENCE)

Exit logic applies only when position is LONG.

PRIMARY EXIT (EXHAUSTION)

Exit LONG immediately when:

RSI(2) > 75

FALLBACK EXIT (TIMEOUT)

Exit LONG when:

Bars held ≥ 4

Precedence rule:
PRIMARY exit must always be evaluated before FALLBACK.
If PRIMARY triggers, FALLBACK must not be used.

EXECUTION RULES

Single position only (no pyramiding)

No short trades

Entry only when flat

Exit resets position to flat

Bars held must be tracked internally

RISK MANAGEMENT

No fixed stop-loss

No ATR-based stop

No trailing stop

Default Trade_Scan risk and sizing rules apply
